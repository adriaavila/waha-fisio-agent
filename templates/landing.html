⚙️ Guía de Configuración Rápida

1

Obtener API Key de Cal.com

Inicia sesión en Cal.com, ve a Settings > Developer > API Keys y genera una clave de acceso (comienza con cal_live_).

2

Configurar el ID del Evento (Servicio)

Identifica el ID de tu servicio de fisioterapia en Cal.com (el ID numérico que deseas que el bot agende de forma predeterminada).

3

Configurar Variables en Coolify

En el panel de Coolify de esta aplicación, navega a Environment Variables y añade las siguientes llaves:

CALCOM_API_KEY="tu_api_key_de_cal.com"
CALCOM_EVENT_TYPE_ID="id_del_evento"
GEMINI_API_KEY="tu_api_key_de_google_gemini" # (Para activar el cerebro de IA)
4

Configurar Webhooks (¡Crucial para sincronización!)

Para que el bot responda al instante cuando alguien agende en Cal.com o escriba por WhatsApp:

  • En WAHA: Registra la URL del webhook apuntando a http://[vps-ip]:8000/webhook/whatsapp para el evento message.
  • En Cal.com: Ve a Webhooks, crea uno nuevo con URL http://[vps-ip]:8000/webhook/calcom y suscríbete a los eventos de Cita Creada y Cita Cancelada.
