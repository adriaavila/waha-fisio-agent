⚙️ Guía de Integración Rápida

Sigue estos 4 pasos para lanzar tu propio bot asistente en tu VPS.

Generar tu API Key en Cal.com

Crea una cuenta en Cal.com si aún no la tienes. Ve al apartado de Settings > Developer > API Keys en tu panel y presiona en crear nueva clave (comienza con cal_live_). Cópiala para configurarla luego en nuestro dashboard.

Obtener tu Event Type ID

Entra a la sección de Event Types en Cal.com y crea el evento de tu consulta de kinesiología (ej: "Kinesiología 1 hora"). Al guardar o editar el evento, fíjate en el número ID que figura en la barra de direcciones del navegador. Ese es el Event Type ID.

Configurar credenciales en el Dashboard de KineLife

Inicia sesión en tu dashboard del agente, ve a las pestañas de Calendario, WhatsApp e IA para guardar tu API Key de Cal.com, tus datos de WAHA y tu key de Gemini o OpenAI. Los cambios se guardarán de inmediato en tu base de datos SQLite.

Activar sincronización por Webhooks (¡Crítico!)

Registra los webhooks para que el bot y el panel se enteren de todo al instante:

  • En WAHA: Configura un webhook apuntando a http://[tu-vps-ip]:8000/webhook/whatsapp para escuchar mensajes entrantes.
  • En Cal.com: Crea un webhook en Cal.com apuntando a http://[tu-vps-ip]:8000/webhook/calcom y suscríbete a Booking Created y Booking Cancelled.

¿Listo para automatizar tu consulta?

Accede al panel del administrador de inmediato y conecta tus servicios. El bot comenzará a responder y agendar en tu WhatsApp al instante.
